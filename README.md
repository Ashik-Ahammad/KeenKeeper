## 📌 Description
Keen Keeper is a modern web application that helps you maintain meaningful relationships by tracking your interactions with friends. It reminds you when to reconnect and visualizes your communication habits through a clean and intuitive interface.

## 🚀 Technologies Used
- NEXT.js
- React
- Context API
- Tailwind CSS
- DaisyUI
- React Icons
- Recharts
- Lucid React

## ⭐ Key Features
1. **📅 Interaction Timeline:** Track all your interactions (Call, Text, Video) with friends in a structured timeline with date and time.
2. **🔔 Smart Status Tracking:** Automatically categorizes friends as *on-track*, *almost due*, or *overdue* based on your interaction frequency.
3. **📊 Visual Statistics:** Displays charts and analytics of your communication patterns using Recharts for better insights.

## 📂 Project Purpose
This project is designed to improve personal relationship management by encouraging consistent communication and providing actionable insights.
